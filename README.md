## 🤖 Работа с AI-ассистентами

Этот шаблон оптимизирован для работы с AI (Cursor, Claude, ChatGPT и др.).

### Как это работает

1. **Положите материалы в `input/`** — черновики, код, данные, источники
2. **Запустите AI-ассистент** и попросите помочь с ВКР
3. **AI использует ваши материалы** — не переписывает, а оформляет

### Гарантии

- 🛡️ **Файлы в `input/` защищены** — AI только читает, не изменяет
- ✍️ **Ваши формулировки сохраняются** — AI использует то, что вы написали
- ⚡ **Экономия времени и токенов** — минимум генерации, максимум переиспользования

### Пример запроса к AI

```
Изучи материалы в input/ и помоги оформить главу 1 (исследовательский раздел).
Используй мои формулировки из input/drafts/глава1.md
```

> Подробные правила для AI находятся в `.cursor/rules/vkr-workflow.mdc`

## 🚀 Быстрый старт

### 1. Установи Typst

```bash
# macOS
brew install typst

# Linux (Arch)
sudo pacman -S typst

# Linux (другие) — через cargo
cargo install typst-cli

# Windows — скачай с https://github.com/typst/typst/releases
```

### 2. Заполни свои данные

Отредактируй `meta.typ`:

```typst
#let meta = (
  student: "Твоё ФИО",
  group: "Твоя группа",
  title: "Тема твоей работы",
  // ...
)
```

### 3. Напиши главы

Редактируй файлы в `chapters/`:

```typst
// chapters/02-research.typ
= Исследовательский раздел

== Характеристика предметной области

Твой текст здесь. Можно писать:
- URL: https://example.com
- Символы: № 123
- Комментарии в тексте: // это работает

Никакого экранирования!
```

### 4. Скомпилируй

```bash
typst compile main.typ

# Или watch-режим
typst watch main.typ
```

## 📖 Как использовать компоненты

### Рисунки

```typst
#import "../template.typ": vkr-image, vkr-placeholder

// Реальное изображение
#vkr-image("../images/arch.png", caption: [Архитектура системы])

// Заглушка
#vkr-placeholder(caption: [Диаграмма], height: 150pt)
```

### Таблицы

```typst
#import "../template.typ": vkr-table, vkr-long-table

// Обычная таблица
#vkr-table(
  columns: (auto, 1fr, auto),
  caption: [Сравнение],
  [*Критерий*], [*Значение 1*], [*Значение 2*],
  [Цена], [100], [200],
)

// Длинная таблица (заголовок повторяется)
#vkr-long-table(
  columns: (auto, 1fr),
  caption: [Результаты],
  header: ([*№*], [*Описание*]),
  [1], [Первый],
  [2], [Второй],
  // ...много строк...
)

// С zebra-полосами
#vkr-table(..., zebra: true)
```

### Листинги кода

```typst
#import "../template.typ": vkr-code

#vkr-code(
  caption: [Пример на Python],
  lang: "python",
  ```
  def hello():
      print("Hello, World!")
  ```.text,
)
```

### Формулы

```typst
// Нумеруются автоматически: (1.1), (1.2)...
$ E = m c^2 $

$ sum_(i=1)^n x_i $
```

## 📚 Библиография

### Вариант 1: YAML файл (рекомендуется)

В `main.typ`:
```typst
#vkr-sources("references.yaml")
```

Формат `references.yaml`:
```yaml
ivanov2024:
  type: Book
  title: "Машинное обучение"
  author: Иванов, Иван Иванович
  date: 2024
  publisher: Издательство
  location: Москва
```

### Вариант 2: Ручной список

```typst
#vkr-sources((
  [Автор. Название. — М.: Издательство, 2024. — 100 с.],
  [Другой автор. Статья // Журнал. — 2024. — № 3.],
))
```
